## Summary

Anthropic shipped 120+ features in 90 days across Claude's product suite, with this article providing a ranked tier list of features most valuable for PMs and builders, along with practical workflows for implementation.

## Key Takeaways

- **Prioritize S-tier features first** - Focus adoption efforts on game-changing features like Claude Code and Cowork that demonstrably change daily workflows before expanding to lower-tier features.
- **Create feature chains for workflows** - Combine multiple Claude features into integrated PM workflows rather than using features in isolation to maximize productivity gains.
- **Build a personalized Claude setup** - Establish your own optimized configuration of Claude features with supporting CLAUDE.md files and prompts tailored to your specific PM responsibilities.
- **Learn feature context before adoption** - Understand the scenarios where each feature delivers measurable value (S-tier weekly, A-tier occasionally) rather than adopting all features broadly.
- **Study Anthropic's product velocity model** - Extract lessons from how Anthropic achieved rapid feature shipping to inform your own product development and release cadence.
