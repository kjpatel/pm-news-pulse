## Summary

This episode with Pawel Huryn provides a practical guide for PMs on when and how to use Claude Chat, Cowork, and Code, demonstrating that Chat is insufficient for serious work while Cowork and Code enable persistent, file-accessing, and self-improving systems.

## Key Takeaways

- **Replace Chat with Cowork or Code** for 95% of PM work—Chat lacks continuity across devices, file access, and persistence, making it a dead end for serious tasks.
- **Build custom Skills iteratively** by installing marketplace baselines, testing on real tasks, gathering specific failure feedback, and rewriting skills based on root causes rather than expecting magic on the first try.
- **Connect MCP servers to your actual tools** (Gmail, Google Drive, Slack, analytics) in draft-only mode first, then let the system learn from your edits to improve drafts over time.
- **Use Cowork for 70% of work and Code for 25%** when you need systems that persist beyond 20 files, custom interfaces, or multi-agent workflows that Cowork's linear chat structure cannot support.
- **Deploy Dispatch agents for 24/7 automation** on scheduled tasks like daily digests or weekly report generation, freeing your real-time sessions for strategic work that requires your judgment.
