## Summary

Claire Vo breaks down five major announcements from Anthropic's 'Code with Claude' event, including scheduled routines, outcome-based agents, multi-agent orchestration, and memory systems, explaining how these features enable builders to create more sophisticated agentic products.

## Key Takeaways

- **Use Claude Code routines** to automate recurring workflows on schedules or webhooks instead of manually triggering processes each time.
- **Implement outcome-based grading** with rubric-based agent evaluation to ensure AI agents are meeting your specific success criteria, not just executing tasks.
- **Build specialized AI teams** using multi-agent orchestration to assign different roles and tools to different agents, increasing coordination and capability.
- **Leverage Dreams memory system** to maintain long-term behavioral context for agents, enabling more consistent and intelligent decision-making over time.
- **Take advantage of increased usage limits** for Claude Code, which unlocks new possibilities for scaling agentic products that were previously constrained.
